## 📄 What is Arcana?

Arcana syncs data from PostgreSQL to your screen in real time. It uses WebSockets to send updates instantly. It works with popular tools like React, Vue, and Svelte. You get an easy way to keep data fresh without refreshing your page.

Arcana fits projects where live updates matter. For example, chat apps, dashboards, or any site that needs fast data changes. It includes tools to track changes, make quick edits, and debug data flows.

## 💻 System Requirements

Arcana runs on Windows computers. Check these before installing:

- **Operating System:** Windows 10 or later (64-bit recommended)  
- **Memory:** At least 4 GB RAM  
- **Storage:** Minimum of 100 MB free space  
- **Internet:** Required for setup and syncing data  
- **Browser:** Modern browsers like Chrome, Firefox, Edge, or Safari

   If you do not have these, ask your system administrator or skip this step for now.

7. **Enjoy Real-Time Sync**

## 🛠 Using Arcana’s Main Features

Arcana makes data syncing easy. Here are some simple actions you can do:

- **View Updates Live**  
  Open your app or site where Arcana syncs data. Changes in your database show up on screen without reloading.

- **Make Changes with Mutations**  
  Arcana allows you to send changes back to PostgreSQL. This lets you edit or add data from the frontend instantly.

- **Use React, Vue, or Svelte Adapters**  
  Your frontends stay connected with minimal setup using ready-to-use adapters. These handle communication with Arcana.

- **Debug with DevTools**  
  Inspect data flow and see all updates in real time. DevTools help spot problems quickly.

- **Handle Offline Mode**  
  Work without a connection. Arcana saves your changes until you reconnect and sync everything.

## 📂 File Structure Overview

When you install Arcana, it creates folders and files:

- `config/`: Settings to connect Arcana to your database  
- `logs/`: Records of activity and errors  
- `adapters/`: Code for React, Vue, and Svelte integration  
- `devtools/`: Debugging tools for data  
- `arcana.exe`: Main program file

## 🔧 Troubleshooting Common Issues

If Arcana does not work as expected, try these:

- Check internet connection  
- Make sure PostgreSQL server is running and accessible  
- Confirm you entered the correct database settings  
- Restart Arcana and your computer  
- Look at the log files for error messages (`logs/` folder)  
- Update Windows and your browser to latest versions
